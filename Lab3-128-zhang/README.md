## 使用步骤

1. **添加个人照片**：
   - 将你的个人照片放在项目根目录
   - 命名为 `photo.jpg`（或修改HTML中的文件路径）
   - 建议使用标准的证件照或清晰的个人照片

2. **修改个人信息**：
   - 在HTML中找到对应的信息字段
   - 修改姓名、性别、年龄、联系方式等内容

3. **本地预览**：
   - 用浏览器直接打开 `index.html` 文件即可预览

4. **云端部署**（可选方案）：
   - **方案A：GitHub Pages**
     1. 将项目上传到GitHub
     2. 在Settings中启用GitHub Pages
     3. 获得部署URL
   
   - **方案B：Netlify**
     1. 访问 https://www.netlify.com
     2. 拖拽项目文件夹上传
     3. 自动获得URL
   
   - **方案C：Vercel**
     1. 访问 https://vercel.com
     2. 导入GitHub仓库或直接部署
     3. 获得URL
