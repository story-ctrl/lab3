## ☁️ 云端部署（30秒快速版）

### 使用 Netlify（最简单）
```
1. 访问 https://app.netlify.com/drop
2. 拖拽整个 Lab3-128-zhang 文件夹进去
3. 30秒内获得URL
4. 完成！
```

### 使用 GitHub Pages
```
1. 新建GitHub账户（如无）
2. 创建仓库名：yourusername.github.io
3. 上传文件
4. 访问 https://yourusername.github.io 即可
```
